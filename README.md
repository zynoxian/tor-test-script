# Tor & I2P Scripts (Windows & Linux)

Simple Python scripts to test Tor and I2P connections.

## Windows
- Requires Tor Browser
- Tor SOCKS proxy: 127.0.0.1:9150
- Scripts located in `/windows`

## Linux
- Requires tor service
- Tor SOCKS proxy: 127.0.0.1:9050
- Scripts located in `/linux`

## I2P
- SOCKS proxy: 127.0.0.1:4447
- Works on both platforms

Purpose:
Educational demonstration of anonymity networks.
